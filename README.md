# The Global Margin

Geopolitical intelligence — unfiltered, ad-free, open.

## Quick Start

1. Get a free GNews API key at https://gnews.io
2. `npm install`
3. Copy `.env.example` to `.env.local` and add your key
4. `npm i -g vercel && vercel dev`
5. Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import at vercel.com
3. Add `GNEWS_API_KEY` in Settings → Environment Variables
4. Deploy
